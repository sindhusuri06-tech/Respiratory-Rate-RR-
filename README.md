# 🩺 SepsisSense AI

## AI-Powered Sepsis Early Warning System

SepsisSense AI is a machine learning-based healthcare application that predicts the risk of sepsis using patient vital signs. The system helps healthcare professionals identify high-risk patients early, enabling faster clinical intervention and improving patient outcomes.

---

## Features

- AI-based sepsis prediction
- User-friendly Streamlit dashboard
- Risk score calculation
- Early warning alerts
- Patient vital signs monitoring
- Machine learning model using Random Forest
- Simple and beginner-friendly implementation

---

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Matplotlib
- Plotly

---

## Dataset

The project uses patient vital signs including:

- Heart Rate
- Body Temperature
- Blood Pressure
- Respiratory Rate
- Oxygen Saturation (SpO₂)
